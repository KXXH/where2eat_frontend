# where2eat_frontend

## 注

实际上本项目是为了解决寝室里三个人一到吃饭时间就不能确定去哪吃饭这个老大难问题而创作的。

## 功能清单

✔通过GitHub pages实现前端的随机抽奖

✔记录吃饭的地点、日期、时间，供后续分析（需后端）

✔简单的账号系统（需后端）

✔简单的时间线回看功能（需后端）

🔨根据历史信息，修改随机数权值

🔨历史记录分析

🔨根据API提供美食推荐
